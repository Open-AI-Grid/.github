# Open AI Grid

Open AI Grid is an open initiative to build a decentralized AI ecosystem.

## Mission

To make artificial intelligence open, transparent and accessible for everyone.

## Vision

A global network where individuals, communities and organizations can contribute compute, models, tools and knowledge.

## Principles

- Open
- Transparent
- Community Driven
- Decentralized

## Status

🚧 Early planning phase

We are currently defining the architecture, governance model and roadmap.
