# Open AI Grid

Open AI Grid is an open initiative to build a global, distributed decentralized AI ecosystem owned bz no one and available to everyone.

## Mission

Open AI Grid aims to democratize access to artificial intelligence by creating a global network of community-operated AI nodes.

## Vision

A global network where individuals, communities and organizations can contribute compute, models, tools and knowledge.

## Principles

- Open
- Transparent
- Community Driven
- Decentralized
- Privacy First
- Distributed by Design
- Accessible to Everyone
